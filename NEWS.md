# panelforest news

## panelforest 0.2.0

- `fp_ci()` / `add_ci()`: added `arrow_length`, `arrow_type`, and `arrow_angle`
  arguments to control the style of truncation arrowheads.
- CI panel (log scale): left-side axis padding is now slightly tighter than the
  right side for a cleaner appearance (`CI_EXPANSION_MULT_LEFT`).

## panelforest 0.1.0

- Added a patchwork-based MVP for tidy forest plots with text, bar, dot, CI, and custom panels.
- Added plot-level structure controls for stripes, summary rows, group-title rows, row-level styling, and horizontal separators.
- Added formatter helpers, built-in themes, bundled example data, README guidance, and package vignettes.
